**Step 3: Interactive REPL (DEMO-05):**
- Only run if `opts.pause !== false` (same pattern as code-search demo)
- Print section header: `pc.cyan('  -- Try it yourself --')`
- Print instructions: "Type any text to see how similar it is to the sample texts."
- Print `/quit` hint
- Create readline interface with prompt `pc.cyan('  nano> ')`
- On each line:
  - Skip empty, handle `/quit`, `/exit`, `/q`
  - Call `generateLocalEmbeddings([input], { dimensions: 1024 })` to embed user text
  - Compare against the 9 cached sample embeddings using `cosineSimilarity()`
  - Sort by score descending
  - Show top 5: `#1  0.XXXX  "sample text here"` format with `pc.bold` for rank, `pc.dim` for score
  - Wrap in try/catch, show `pc.yellow('Warning')` on error
- On close, continue to Step 4

**Step 4: Shared Embedding Space Proof (DEMO-06):**
- Detect API key: `process.env.VOYAGE_API_KEY || getConfigValue('apiKey')`
- If no API key: skip silently (no message about missing key -- the demo is zero-dependency)
- If API key found:
  - Print: `pc.cyan('  -- Bonus: Shared Embedding Space --')`
  - Print: "API key detected -- comparing nano vs API rankings."
  - theory(verbose, ...) explaining shared embedding space
  - Pick 3 query texts (one per cluster): use sample texts at indices 0, 3, 6 (first from each cluster)
  - For each query:
    - Nano rankings: already have cached embeddings, compute similarity of query against all 9, sort, take top 5
    - API rankings: call `generateEmbeddings([queryText], { model: 'voyage-4-large', inputType: 'query' })`, then call `generateEmbeddings(sampleTexts, { model: 'voyage-4-large', inputType: 'document' })` (cache the document embeddings after first query to avoid redundant API calls), compute similarity, sort, take top 5
    - Display side-by-side: two columns showing rank + label for nano and API
    - Highlight where rankings match
  - Wrap ALL API calls in try/catch. On any failure: print `pc.dim('  API comparison unavailable: ' + err.message)` and continue to next steps. Never crash on the optional bonus.
