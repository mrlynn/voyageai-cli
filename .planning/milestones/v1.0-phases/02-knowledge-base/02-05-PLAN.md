Create src/nano/nano-setup.js with step-based setup and clear-cache functionality.

**Constants:**
```javascript
const VENV_DIR = path.join(os.homedir(), '.vai', 'nano-env');
const VENV_PYTHON = path.join(VENV_DIR, process.platform === 'win32' ? 'Scripts' : 'bin', process.platform === 'win32' ? 'python.exe' : 'python3');
const MODEL_CACHE_DIR = path.join(os.homedir(), '.vai', 'nano-model');
const REQUIREMENTS_PATH = path.join(__dirname, 'requirements.txt');
```

**detectPython():**
- Try `python3 --version` then `python --version` using execFileSync with 5s timeout
- Parse output with regex `/Python\s+(\d+)\.(\d+)\.(\d+)/`
- Require major >= 3 && minor >= 10
- Return `{ command, version, fullVersion }` on success
- Throw `createNanoError('NANO_PYTHON_NOT_FOUND')` if no suitable Python found
- Use `{ encoding: 'utf8', stdio: 'pipe' }` options

**createVenv(pythonCmd):**
- Ensure `~/.vai/` exists with `fs.mkdirSync(path.join(os.homedir(), '.vai'), { recursive: true })`
- Run `execFileSync(pythonCmd, ['-m', 'venv', VENV_DIR], { timeout: 30_000, stdio: 'pipe' })`
- Verify VENV_PYTHON exists after creation
- Throw descriptive error if venv creation fails

**installDeps(pythonCmd is not needed -- use VENV_PYTHON):**
- Build pip args: `['-m', 'pip', 'install', '-r', REQUIREMENTS_PATH, '--quiet']`
- Platform-aware PyTorch: on Linux, check for nvidia-smi. If not found, add `'--extra-index-url', 'https://download.pytorch.org/whl/cpu'` to args
- Use `spawn` (not execFileSync) with `stdio: ['ignore', 'inherit', 'inherit']` so pip output streams to terminal in real-time
- Return a Promise that resolves on exit code 0, rejects otherwise

**downloadModel():**
- Python script to execute in venv:
  ```python
  import os
  os.environ['SENTENCE_TRANSFORMERS_HOME'] = '<MODEL_CACHE_DIR>'
  from sentence_transformers import SentenceTransformer
  model = SentenceTransformer('voyageai/voyage-4-nano', trust_remote_code=True, cache_folder='<MODEL_CACHE_DIR>')
  print('OK')
  ```
- Use `spawn(VENV_PYTHON, ['-c', script])` with stderr piped (HuggingFace download progress)
- Stream stderr to terminal for download progress visibility
- Return Promise, reject with `createNanoError('NANO_MODEL_NOT_FOUND')` on failure

**Step check functions:**
- `checkVenvExists()`: return `fs.existsSync(VENV_PYTHON)`
- `checkDepsInstalled()`: run `execFileSync(VENV_PYTHON, ['-c', 'import sentence_transformers; import torch; print("ok")'], { encoding: 'utf8', timeout: 10_000, stdio: 'pipe' })` -- return true if exit 0, false otherwise
- `checkModelExists()`: check `fs.existsSync(MODEL_CACHE_DIR)` AND directory has a subdirectory containing 'voyage' in name

**STEPS array:**
```javascript
const STEPS = [
  { name: 'Detecting Python 3.10+', fn: detectPython, check: null },
  { name: 'Creating virtual environment', fn: createVenv, check: checkVenvExists },
  { name: 'Installing Python dependencies', fn: installDeps, check: checkDepsInstalled },
  { name: 'Downloading voyage-4-nano model', fn: downloadModel, check: checkModelExists },
];
```

**runSetup(options = {}):**
- Iterate STEPS. If `!options.force && step.check && step.check()`, log skip and continue.
- If `options.force`: delete VENV_DIR and MODEL_CACHE_DIR before starting (using `fs.rmSync(dir, { recursive: true, force: true })`)
- For each step: show spinner via ui.spinner(), call step.fn(), succeed/fail spinner
- detectPython returns the python command, pass it to createVenv step. After that, all steps use VENV_PYTHON.
- On completion: log success summary with total elapsed time

**runClearCache(options = {}):**
- If MODEL_CACHE_DIR doesn't exist: log "No cached model files found." and return
- Calculate directory size (recursive walk, sum file sizes) and display it
- If `!options.yes`: prompt with readline for y/N confirmation
- On confirm: `fs.rmSync(MODEL_CACHE_DIR, { recursive: true, force: true })`, log success
- On cancel: log "Cancelled."

**getDirSize(dirPath):** Helper to recursively sum file sizes for display.

**Exports:** `{ runSetup, runClearCache, detectPython, checkVenvExists, checkDepsInstalled, checkModelExists, VENV_DIR, VENV_PYTHON, MODEL_CACHE_DIR }`

Export individual functions and constants for testability (TEST-03 will mock these).

```javascript
'use strict';

function registerNano(program) {
  const nano = program
    .command('nano')
    .description('Local inference with voyage-4-nano');

  nano
    .command('setup')
    .description('Set up local inference environment (Python venv, deps, model)')
    .option('--force', 'Rebuild environment from scratch')
    .action(async (options) => {
      const { runSetup } = require('../nano/nano-setup.js');
      await runSetup(options);
    });

  nano
    .command('clear-cache')
    .description('Remove cached model files')
    .option('--yes', 'Skip confirmation prompt')
    .action(async (options) => {
      const { runClearCache } = require('../nano/nano-setup.js');
      await runClearCache(options);
    });

  // status, test, info subcommands will be added in Plan 02-06

    .command('test')

    .command('info')

}

module.exports = { registerNano };
```
